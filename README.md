 # Proyecto Sorteador Realizado en Python 

La App utiliza el moudul tkinter, y la libreria pillow

Como Correr el proyecto

Creamos el entorno virtual

py -m venv .venv

-- Activamos el entorno virtual 

source .venv/bin/activate

pip install -r requirements.txt

-- Verificamos si tenemos el modulo tkinter 

  py -m tkinter

-- Instalacion modulo tkinter

 sudo apt update 

 sudo apt install python3-tk
